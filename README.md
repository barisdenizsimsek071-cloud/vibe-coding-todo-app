# Todo List App (Vibe Coding with Replit Agent)

A simple Todo List web application built as part of a homework assignment on vibe coding tools. The app allows users to add, edit, and delete tasks, with persistence via localStorage in the browser.

## Features
- Add new tasks
- Edit existing tasks
- Delete tasks
- Persist tasks with localStorage

## Technologies Used
- HTML
- CSS
- JavaScript
- Replit Agent (as the vibe coding tool)

## Setup and Installation
1. Clone the repository
2. Open `index.html` in your browser

No dependencies required.

## How to Run
- Double-click `index.html` to open it in a web browser.
